/* CSS Box Model: Every HTML element is treated as a rectangular box. The box model consists of: 1) Content – The actual text or image inside the element. 2) Padding – Space between content and border. 3) Border – The outline around padding and content. 4) Margin – Space outside the border (distance from other elements). Total width = content width + padding + border + margin Total height = content height + padding + border + margin By default, width and height apply only to the content. If we use: box-sizing: border-box; then padding and border are included inside the given width and height. The box model helps control spacing and layout of elements properly. */
